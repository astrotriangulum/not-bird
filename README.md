# BirbyTaby
The birb tab (BirbyTaby) is a simple custom new tab page designed to make opening a new tab a little more fun and cozy.

## Live Demo
Try it: https://glowing-kelpie-95e187.netlify.app/

---

## Features
* **Live Clock**
* **Current Date**
* **Time-Based Greeting**
* **Working Search Bar**
* **Notepad**
* **Apps Shortcuts**
* **Floating Birds (in a line)**

## Coding Languages Used
* **HTML5**
* **CSS3**
* **Javascript**

---

## Local Running Guide
1. Clone the repository (ofc).
2. Open the project folder.
3. Double click 'index.html' or just open it in your web browser.
